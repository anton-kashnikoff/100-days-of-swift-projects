## Description
A memory pairs game that has players find pairs of cards.
## System requirements
* Swift 5.9
* iOS 13.0+
## Technologies
* UICollectionViewController, UICollectionViewLayout, UICollectionViewCell
* UITableView, UITableViewCell
* UIStoryboard
* GCD (Grand Central Dispatch)
